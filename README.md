# Thesis: Calibration Evaluation of Molecular Docking Models

MPhil thesis project evaluating whether DiffDock and SigmaDock produce well-calibrated pose distributions using the TARP and MIRA calibration metrics.

## What this repo does

Takes pre-run inference results from DiffDock (and SigmaDock) and evaluates them on three test sets:

- **PDBBind test set** — 322 protein-ligand complexes, 40 predicted poses each
- **PoseBusters benchmark** — 305 complexes, 40 poses each, with physical validity filtering
- **Top-1 runs** — 10 independent DiffDock runs merged into top-1 (10 poses) and top-3 (30 poses) variants

Metrics computed:
- **TARP** (Tests of Accuracy with Random Points) — calibration diagnostic via ECP curves
- **MIRA** (Model-Independent Random-radius Assessment) — scalar calibration score, null ≈ 0.683 for S=40
- **RMSD accuracy** — fraction of complexes with top-1 pose within 2Å / 5Å of crystal
- **PoseBusters pass rate** — fraction of poses passing ~20 physical validity checks

## Repository layout

```
molcalib/           Core metric implementations (installable package)
  mira.py           MIRA score (symRMSD-based, Sharief et al. 2026)
  tarp.py           TARP fractions and ECP plotting
  distances.py      Symmetry-corrected RMSD via spyrmsd (with timeout)
  prior.py          Reference pose prior (generate_reference_coords)

  loader.py         Load crystal coords, DiffDock rank*.sdf outputs, Cα coords
  tarp_runner.py    Parallel TARP evaluation over a test set
  mira_runner.py    Parallel MIRA evaluation over a test set
  pb_eval.py        PoseBusters filtering and PB-filtered MIRA/RMSD
  scripts/          Entry-point scripts called by SLURM jobs

eval_sigmadock/     SigmaDock-specific evaluation (mirrors eval_diffdock)

notebooks/          Analysis and figure generation
  tarp_analysis.ipynb              TARP ECP curves + protein-family breakdown
  posebusters_calibration.ipynb    TARP/MIRA on the PoseBusters benchmark
  pb_filtering_analysis.ipynb      Root-cause breakdown of PB pass/fail
  group_tarp_mira.ipynb            Per-protein-family calibration comparison
  confidence_calibration_analysis.ipynb  Confidence score vs calibration quality
  distribution_analysis.ipynb      Pose distribution visualisations

figures/            Output figures (generated by notebooks, not tracked)
logs/               SLURM job stdout/stderr
diffdock/           DiffDock source (submodule / external)
sigmadock/          SigmaDock source (submodule / external)
```

## Data

All large data lives on RDS at `~/rds/hpc-work/`:

```
data/
  PDBBind_processed/          Crystal structures + processed proteins
  posebusters_benchmark_set/  PoseBusters crystal structures
  inference/                  Inference CSVs and protein annotations

results/DiffDock/
  pdbbind_testset/            40-sample baseline inference results
  pb_evaluate_v2_merged/      PoseBusters inference results (poses/ subdir)
  top1_runs_v2_top1_merged/   10-run top-1 merged (10 poses per complex)
  top1_runs_v2_top3_merged/   10-run top-3 merged (30 poses per complex)
```

Each results directory contains a `metrics/` subdirectory with pre-computed `.npy` arrays (TARP fractions, MIRA scores, RMSD arrays, etc.) that the notebooks load directly.

## Running evaluations

All heavy computation runs via SLURM (Cambridge HPC, icelake partition). Scripts are in `~/slurm/thesis/`:

```bash
# TARP + MIRA for PDBBind test set
sbatch ~/slurm/thesis/run_tarp_mira_pdbbind.sh

# TARP + MIRA for PoseBusters benchmark
sbatch ~/slurm/thesis/run_tarp_mira_pb_benchmark.sh

# TARP + MIRA for top-1/top-3 merged runs (pass --variant top1 or top3)
sbatch ~/slurm/thesis/run_tarp_mira_top1runs.sh

# Run a specific analysis notebook
sbatch ~/slurm/thesis/run_notebook.sh notebooks/tarp_analysis.ipynb
```

Logs are written to `logs/<jobname>_<jobid>.{out,err}`.

## molcalib package

The `molcalib` package contains the metric implementations and can be installed for use outside this repo:

```bash
pip install -e .
```

Key functions:
- `molcalib.mira.mira_score(...)` — MIRA score for one complex (symRMSD)
- `molcalib.mira.mira_null(S)` — null reference score for S samples
- `molcalib.tarp.tarp_fractions(...)` — TARP fractions for one complex
- `molcalib.tarp.ecp_from_fractions(...)` — ECP curve from TARP fractions
- `molcalib.distances.compute_rmsd_symmetry(...)` — symmetry-corrected RMSD (spyrmsd, 4s timeout)
